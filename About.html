About The Razer Rage Mode Project:

This Project was made to show off various techniques that can be utilised with the use of HTML, CSS and also Javascript. This website was made for Assignment 3 of the web authoring module.

The Images:

All of the images used were manipulated with the use of photoshop, and the originals are a product of the company known as razer. the majority of the images used featured a wide array of various colours that I removed and replaced them with the shade of red that can be seen.
